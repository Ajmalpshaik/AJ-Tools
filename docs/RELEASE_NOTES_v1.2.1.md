# AJ Tools v1.2.1 Release Notes

Release date: 2026-04-07

## Changes
- Updated About tool to a dedicated About window with:
  - current version display
  - updated feature highlights
  - clickable LinkedIn and email links
  - optional profile photo support from `Resources/AboutPhoto.png`
- Added profile image resource support in release payload (`AboutPhoto.png`).

## Included In Release Package
- `AJ Tools.dll`
- `Newtonsoft.Json.dll`
- `Resources/` (including `AboutPhoto.png`)
- `AJ Tools.addin`
- `install.cmd`, `install-all-users.cmd`, `install.ps1`
- `uninstall.cmd`, `uninstall-all-users.cmd`, `uninstall.ps1`

## Installation
1. Download `AJ-Tools-v1.2.1.zip` from GitHub Releases.
2. Extract the zip.
3. Run `install.cmd` for current-user install.
4. For all-users install, run `install-all-users.cmd` as Administrator.
