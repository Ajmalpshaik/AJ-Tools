# AJ Tools v1.2.0 Release Notes

Release date: 2026-04-07

## Highlights
- Added new `Smart MEP Tag` tool (smart placement workflow with per-category settings).
- Added new `Arrange Tags` tool with intelligent vertical stacking logic.
- Added shared `LeaderLogicService` and updated `L-Shape Leader` behavior.
- Smart leader elbow handling improved to keep elbow outside tag text bounds.
- Smart tag offset logic updated to use element-to-tag-text-edge distance.
- Added/updated ribbon icons for Smart MEP Tag and Arrange Tags.

## Included In Release Package
- `AJ Tools.dll`
- `Newtonsoft.Json.dll`
- `Resources/` (icons and assets)
- `AJ Tools.addin`
- `install.cmd`, `install-all-users.cmd`
- `install.ps1`
- `uninstall.cmd`, `uninstall-all-users.cmd`
- `uninstall.ps1`

## Installation
1. Download `AJ-Tools-v1.2.0.zip` from https://github.com/Ajmalpshaik/AJ-Tools-Installer/releases
2. Extract the zip.
3. Run `install.cmd` for current-user install.
4. For all-users install, run `install-all-users.cmd` as Administrator.
